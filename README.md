# Tic Tac Toe

This project makes a tic tac toe game using basic python and tkinter library for GUI.

## Installation

Just have a working version of python installed.
Might have to install the tkinter module.

```bash
pip install tkinter
```

## Files

The script named tic_tac_toe.py is not linked to any GUI and just takes input from terminal.

ttt.py is the actual script which includes GUI
